# Pacote: Humanizador de Escrita de IA

Este ZIP contém uma skill completa para humanização de textos com aparência de IA.

## Conteúdo
- `index.html` — Super Prompt pronto para uso
- `SKILL.md` — Documentação obrigatória da skill
- `README.md` — Instruções gerais

## Como usar
1. Abra o arquivo `index.html` no navegador.
2. Cole o texto que deseja humanizar.
3. Siga o fluxo descrito no próprio prompt.

## Requisitos
Nenhum software adicional é necessário.  
Funciona em qualquer navegador moderno.

## Observação
Este ZIP **deve conter** o arquivo `SKILL.md` para ser válido.
